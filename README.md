## ✨ Features

- ✅ **Action item extraction** — every commitment pulled out with owner, deadline, and context
- 🏛️ **Decision log** — key decisions documented so they do not get relitigated
- 📅 **Next steps summary** — clear list of what happens after the meeting ends
- 🙋 **Owner assignment** — AI infers who owns each task from the conversation context
- 📋 **Formatted output** — ready-to-paste Slack message, email, or Notion page
- 🔄 **Supports raw notes and transcripts** — bullet points, paragraphs, or full transcripts all work
- ⚡ **Free, no signup** — paste your notes, get the structured output instantly

## 🚀 Quick Start

```bash
git clone https://github.com/jayblast-spec/meetingmind.git
cd meetingmind
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
