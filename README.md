# MeetingMind

MeetingMind is part of ArkNet Digital's AI Productivity Suite.

## Promise

prepare, capture, and follow up on meetings with memory that compounds

## Built For

sales teams, founders, recruiters, and client-facing operators

## Product Modules

- Attendee context
- Agenda builder
- Decision capture
- Follow-up draft
- Relationship memory

## Open Source Direction

This repo is open to contributors because strong ideas move faster when serious builders can inspect, improve, and extend them. The goal is not secrecy. The goal is speed, quality, and contribution.

## First Workflow

1. Enter real context.
2. Generate a useful first output.
3. Review risk, confidence, or next action.
4. Save the result into product memory in a future paid workspace.

## Run Locally

```bash
npm install
npm run dev
```
